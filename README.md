## 安装运行

```bash
# 安装gitbook
$npm install gitbook -g
$npm install -g gitbook-cli
--- 如果没有安装gitbook,此命令会默认同时安装 GitBook
$gitbook -V
--- 列出本地所有的gitbook版本
$gitbook ls

## 安装依赖
$sh install_all_module.sh

# 运行gitbook&发布
$sh deploy.sh

## push源代码
$git add .
$git commit - m "提交内容注释"
$git push

```
